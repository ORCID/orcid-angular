Change these values to preview `orcid-step-view` behavior:

Title Subtitle Primary action label Secondary action label Disable primary action Full width actions

Use this projected body region for form controls and explanatory text.

The parent flow controls step transitions and handles output events. 

<orcid-step-view
  [title]="title"
  [subtitle]="subtitle"
  [primaryLabel]="primaryLabel"
  [secondaryLabel]="secondaryLabel"
  [primaryDisabled]="isPrimaryDisabled"
  (primaryAction)="onPrimary()"
  (secondaryAction)="onSecondary()"
>
  <div>Step content goes here</div>

  <div orcidStepViewFooter>
    <button mat-button type="button">Extra footer control</button>
  </div>
</orcid-step-view>

Inputs

  • title: Main step title.
  • subtitle: Optional secondary line (for step count copy).
  • primaryLabel: Optional primary button label.
  • secondaryLabel: Optional secondary button label.
  • primaryDisabled: Disables primary action.
  • fullWidthActions: Makes action buttons stretch to width.

Outputs

  • primaryAction: Emitted when primary action is clicked.
  • secondaryAction: Emitted when secondary action is clicked.

Slots

  • Default slot: Main step body content.
  • [orcidStepViewFooter]: Optional projected footer area.
