Typography

Typography tokens from @orcid/tokens. Prefer these CSS variables (or equivalent Tailwind theme keys) over hardcoded font sizes, weights, and line heights so the UI stays aligned with the design system.

Usage

Use semantic tokens (e.g. var(--orcid-font-size-body), var(--orcid-line-height-body)) in component styles instead of raw values. This keeps typography consistent and makes it easier to align with Figma once design tokens are introduced there.
