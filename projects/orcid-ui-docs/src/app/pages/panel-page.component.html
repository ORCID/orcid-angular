  • panelId: Unique identifier for the panel
  • title: Optional title text displayed in the header
  • icon: Optional Material icon name to display in the header
  • iconClass: Optional CSS class for icon styling
  • isMobile: Boolean for mobile-optimized layout
  • isPreferred: Boolean to show highlighted border around the panel
  • startToggleOn: Boolean indicating if the start toggle is ON (star filled)
  • enableStartToggl: Boolean to enable the featured toggle button
  • startToggleDisabled: Boolean to disable the featured toggle button
  • startToggleTooltip: String tooltip text for the featured toggle button

Outputs

  • toggleFeatured: Emitted when the featured toggle button is clicked

Content Projection Slots

  • Default slot: Main panel content
  • panel-footer: Footer area for sources or actions
