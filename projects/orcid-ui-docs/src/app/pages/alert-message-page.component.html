Alert Message

The <app-alert-message> component displays contextual notices, informational messages, warnings, and success confirmations with a consistent layout and iconography.

Variants

Select a variant to preview:

Notice Info Warning Success {{ type | titlecase }} example This is a {{ type }} alert. Use it to communicate contextual feedback to the user.

Usage

Project templates provide two projected slots: [title] and [content].

<app-alert-message type="warning">
  <h2 title>API limit approaching</h2>
  <p content>You have used 85% of your monthly allocation.</p>
</app-alert-message>

Inputs

  • type: 'notice' | 'info' | 'warning' | 'success' (default 'notice')

Accessibility

Icons are presentational. Provide semantic structure in the projected [title] and [content] nodes (e.g. headings, lists).
